## 🚀 Setting Up Your First Task

The application uses the concept of jobs to manage your data. A job connects two folders together. 

1. Launch GoodSync-2026 from your desktop icon.
2. Click the Job menu and select New.
3. Select the folder type. You can choose from local folders, network drives, or popular cloud services.
4. Name your job. 
5. Select the left folder. This is the source folder containing your original files.
6. Select the right folder. This is the destination folder where you want your copies to live.
7. Click the Analyze button. The program shows you which files differ between the two folders.
8. Click the Sync button. The program transfers the files to match the destination folder to the source folder.

4. Set the timer to run the sync every hour, day, or week.
5. Save your changes.

The program runs in the background. It checks for new files and updates your destination folders without any input from you.

## 🔍 Troubleshooting Common Issues

If you run into issues, check these steps first.

*   **The program does not start:** Check that your Windows version is up to date. You might need to restart your computer.

*   **Permission errors:** Ensure you have read and write access to the folders you selected.
